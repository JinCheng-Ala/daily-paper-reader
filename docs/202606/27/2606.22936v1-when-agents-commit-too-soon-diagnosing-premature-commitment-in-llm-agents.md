## 摘要
长程LLM智能体会悄然失败：它们过早地确定了对证据的一种解读，然后在剩余的运行时间里一直维护这一解读。我们将此称为早期承诺。最终答案评分无法捕捉这种失败模式，因为它只看到答案，而不关注过程是否已经坍缩到一条稳定路径。我们将表征承诺定义为在固定推理步骤上跨运行隐藏状态的收敛，并将其用作轨迹一致性的早期诊断。在Llama-3.1-70B上运行ReAct于HotpotQA时，第4步隐藏状态相似性可预测下游行为一致性（r = -0.35，偏相关r = -0.45），并具有局部时间和层级特征。该信号在Qwen-2.5-72B和Phi-3-14B以及StrategyQA（r = -0.83）上得到重复验证。它不追踪正确性：承诺错误和承诺正确的问题在激活相似性上不可区分。这一边界是主张的核心。承诺告诉我们智能体是否已经确定，而非是否正确。一个运行时监控器可从隐藏状态中检测不一致轨迹，AUROC高达0.97（更严格划分下为0.85–0.88），而一个提示干预将行为方差降低了28%（与标记匹配的对照相比），同时准确性在统计上无变化。我们还测试了该信号是否能引导自一致性计算；在一个更难的基准上，它仅提供适度帮助，且被一个更简单的基于输出的基线所匹配。结果是对隐藏过程失败的诊断，具有明确的局限性，而非一个通用准确性杠杆。

## Abstract
Long-horizon LLM agents can fail quietly: they settle on one reading of the evidence early, then spend the rest of the run defending it. We call this premature commitment. Final-answer scoring misses the failure mode because it sees only the answer, not whether the process has already collapsed to a stable path. We define representational commitment as cross-run hidden-state convergence at a fixed reasoning step, and use it as an early diagnostic of trajectory consistency. On Llama-3.1-70B running ReAct on HotpotQA, step-4 hidden-state similarity predicts downstream behavioral consistency (r = -0.35, partial r = -0.45), with a localized temporal and layer-wise signature. The signal replicates across Qwen-2.5-72B and Phi-3-14B, and on StrategyQA (r = -0.83). It does not track correctness: committed-wrong and committed-correct questions are not separable in activation similarity. That boundary is central to the claim. Commitment tells us whether an agent has settled, not whether it is right. A runtime monitor detects inconsistent trajectories from hidden states at AUROC up to 0.97 (0.85--0.88 under a stricter split), and a prompting intervention cuts behavioral variance by 28% against a token-matched control while leaving accuracy statistically unchanged. We also test whether the signal can route self-consistency compute; on a harder benchmark it helps only modestly and is matched by a simpler output-based baseline. The result is a diagnostic for a hidden process failure, with clear limits rather than a general accuracy lever.

## 1. 核心问题与整体含义（研究动机与背景）

- **研究动机**：长程LLM智能体（如使用ReAct框架进行多步检索、推理的智能体）在运行过程中可能**过早承诺**（premature commitment）——即在早期阶段就锁定对某个证据的单一解读，之后的所有行为都变为维护这一立场，而非继续开放地寻求证据。这种失败模式是**隐蔽的**：最终答案看起来可能合理，但过程已经坍缩。
- **为什么重要**：传统的最终答案评分无法检测这一失败，因为它只看输出结果；交叉运行的一致性检验也含糊不清——一个错误但自信的智能体可能与自己一致得很好。因此需要一种**过程级诊断**，能够判断智能体是否**已经稳定（settled）**，而不管该稳定是对还是错。
- **核心构念**：作者提出**表征承诺**（representational commitment），即跨不同运行在固定推理步骤上隐藏状态的收敛程度。高收敛意味着智能体已经“安顿”于一个解释，低收敛意味着它仍对不同的证据路径敏感。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：在智能体的每个推理步骤（step，定义为Thought→Action→Observation三元组），提取最后一层最后一个token的隐藏状态，并计算多个运行（同一问题、非零温度）之间该隐藏状态的**平均成对余弦相似度**。高相似度表示表征收敛，即承诺。
- **关键定义**：
  - **激活相似性**（activation similarity）：公式 (1)：  
    `Sim_q^(s,l) = binom(n,2)^{-1} ∑_{i<j} cos(h_i^(s,l), h_j^(s,l))`  
    其中n为有该步骤隐藏状态的运行数。
  - **行为一致性**（trajectory consistency）：使用步数的变异系数（CV，标准差/均值）和动作序列多样性两个指标。
- **提取时机**：主要分析在第4步（step 4），监控器使用该步的隐藏状态。
- **运行时监控**：训练一个逻辑回归分类器，输入步骤4的隐藏状态特征（如逐层相似性向量），输出“一致”或“不一致”标签（通过交叉验证）。
- **干预实验**：在第3步添加提示指令，要求智能体“承诺”一个推理策略；对照条件包括标准ReAct和填充了等长中性文本的控制。
- **可解释性**：通过主成分分析发现承诺方向接近第一主成分，且与困难度差异向量高度对齐（cos=0.95）。

## 6. 主要结论与发现

1. **承诺信号存在**：步骤4的激活相似性与行为一致性负相关（Llama: r=-0.35, 偏r=-0.45），分层集中在中高层（32-80层），时间集中在步骤3-5，峰值为步骤4。跨架构和跨基准均重复（Qwen: r=-0.65; Phi-3: r=-0.36; StrategyQA: r=-0.83）。
2. **承诺不等于正确**：承诺错误和承诺正确的问题在激活相似性上不可区分（Llama: 0.935 vs 0.903, p=0.30）。承诺反映的是过程稳定与否，而非结果对错。
3. **运行时监控有效**：使用步骤4逐层相似性向量的逻辑回归分类器AUROC高达0.97（分位数标签），在严格中位数划分下为0.85-0.88；当只有3次运行时仍达0.81。可提前退出节省29%计算。
4. **提示干预降低方差**：在第3步添加承诺提示后，行为变异系数降低28%（相对于填充对照，d=0.33, p=0.001），但准确率无显著变化。这符合承诺放大当前轨迹的预期。
5. **路由计算效果有限**：隐藏状态信号在MuSiQue上对路由自一致性计算只有适度帮助，被更简单的输出级自适应一致性基线匹配或超越。

## 7. 优点

- **诊断隐蔽失败**：首次识别并命名了LLM智能体中的“过早承诺”失败模式，并提供了可操作的度量。
- **过程级视角**：超越最终答案和输出一致性，直接测量内部表征的收敛性。
- **跨模型/基准泛化**：在三种不同架构（Llama、Qwen、Phi）和两个推理基准上重复验证，信号稳健。
- **严格统计控制**：使用置换检验、Bonferroni校正、偏相关、多重比较校正、探索/验证分离。
- **诚实报告局限性**：明确指出承诺不区分正确性、路由效果不如简单基线、单层转向失效、检索重叠未完全排除等。
- **实用监控工具**：基于隐藏状态的运行时监控器可在低运行预算下有效，并支持早期退出节省计算。

## 8. 不足与局限

- **检索重叠混淆**：文档重叠部分解释了自然观察中的相关性。尽管部分控制了文档身份，但完全分离推理与检索仍需未来实验（如重放固定文档）。
- **因果证据有限**：中介分析显示间接路径显著但为抑制性中介，直接路径非显著且方向相反。提示干预的机制尚不完全清楚。
- **单层转向实验失败**：尝试通过激活转向控制承诺，但结果混合，说明承诺是多层、多步分布的现象。
- **温度固定**：所有实验仅使用T=0.5，未进行温度扫描。
- **任务覆盖窄**：仅验证了多跳问答和隐含推理任务，代码、数学、具身任务未测试。
- **行为代理粗**：CV和动作多样性是粗糙的代理，可能无法捕捉细粒度轨迹差异。
- **输出路由不足**：信号在更难的基准上（MuSiQue）未能击败简单输出基线，限制了其作为通用计算路由器的应用。
- **假设忠实性**：假设隐藏状态忠实反映了推理过程，但链式思考可能不忠实；论文声明该预测关系而非解释关系。
