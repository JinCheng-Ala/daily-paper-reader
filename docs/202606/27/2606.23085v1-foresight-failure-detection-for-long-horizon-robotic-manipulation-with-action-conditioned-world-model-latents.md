## 摘要
长程任务在真实世界机器人部署中很常见，但此类任务的故障检测仍未被充分探索。检测长程机器人任务中的故障特别具有挑战性，因为故障的起始往往是模糊的，且通常缺乏密集的时间标注。我们提出Foresight，一个故障检测框架，它使用来自动作条件世界模型的潜在表示来监测操作轨迹。Foresight仅使用最终的任务级成功或失败标签进行训练。通过利用预测性世界模型嵌入，我们的方法为跨不同策略的故障检测提供了统一框架。我们进一步使用函数共形预测（FCP）自适应地校准检测阈值。我们在LIBERO-Long、ManiSkill-Long和BEHAVIOR-1K仿真环境中使用最先进的视觉-语言-动作策略评估Foresight，与最先进的故障检测方法进行比较，并在真实机器人上使用ReactorX-200机械臂的三个长程任务和Franka机械臂的一个任务进行验证。我们的结果表明，动作条件世界模型嵌入为长程操作中的可靠故障监控提供了可扩展的表示。

## Abstract
Long-horizon tasks are common in real-world robotic deployments, yet failure detection for such tasks remains underexplored. Detecting failures in long-horizon robotic tasks is particularly challenging because failure onset is often ambiguous and dense temporal annotations are typically unavailable. We present Foresight, a failure detection framework that monitors manipulation trajectories using latent representations from an action-conditioned world model. Foresight is trained using only final task-level success or failure labels. By leveraging predictive world-model embeddings, our method provides a unified framework for failure detection across different policies. We further use functional conformal prediction (FCP) to calibrate detection thresholds adaptively. We evaluate Foresight with state-of-the-art vision-language-action policies in simulation on LIBERO-Long, ManiSkill-Long, and BEHAVIOR-1K, compare it against state-of-the-artfailure detection methods, and validate it on real robots with three long-horizon tasks on a ReactorX-200 arm and one task on a Franka arm. Our results suggest that action-conditioned world-model embeddings provide a scalable representation for reliable failure monitoring in long-horizon manipulation.

- **问题背景**：长程机器人操作任务（如多步骤抓取、放置、开关等）在实际部署中很常见，但现有故障检测方法多聚焦于短周期、单步异常或策略特定置信度，难以处理长周期中故障起始模糊、缺乏密集时间标注的挑战。

## 2. 方法论

- **核心思想**：利用动作条件世界模型（V-JEPA 2-AC）的预测潜变量作为故障检测的特征。世界模型能编码任务相关的状态线索（空间关系、运动模式、交互动态等），其预测潜变量反映了在给定策略动作下场景应如何演变，从而捕捉执行偏差。
- **关键技术细节**：
  1. 冻结预训练的V-JEPA 2-AC视觉编码器，从头训练动作条件预测器（AC predictor）。
  2. 在每个时间步t，当前观测上下文\(c_t\)和策略预测的动作块\(A_t\)输入世界模型，得到**隐藏潜变量**\(z_h^t\)（描述当前观察）和**动作条件预测潜变量**\(z_p^t\)（描述期望的未来状态）。论文使用\(z_p^t\)作为时间步token\(u_t = W z_p^t + p_t\)（\(p_t\)为正弦位置编码）。
  3. 将历史token序列\(U_{\le t}\)输入**因果Transformer**（或LSTM/MLP），输出每步故障分数\(s_t \in [0,1]\)。训练使用轨迹级标签（每步继承，早期检测加权）。
- **阈值校准**：采用**功能共形预测（FCP）** 在验证集成功轨迹上构造时变阈值\(\delta_t = \mu_t + \hat{q}\sigma_t\)，其中\(\mu_t\)为平均分数曲线，\(\sigma_t\)为波动项，\(\hat{q}\)为校准集非一致性分数的\((1-\alpha)\)分位数。当\(s_t \ge \delta_t\)时触发告警，保证false positive rate受控于\(\alpha\)。

- **无需策略内部状态**：仅需观测图像和策略动作块，不依赖策略logits、隐藏状态或token概率，因此可跨不同VLA策略推广。

## 6. 主要结论与发现

1. **Foresight-Transformer在长周期任务上显著优于所有基线**：在LIBERO-Long上平衡准确率0.94，ManiSkill-Long上0.80，BEHAVIOR-1K上0.78，相比最佳基线分别提升6~14个百分点。在最长基准BEHAVIOR-1K上，Foresight-Transformer ROC-AUC达0.76，优于SAFE-LSTM的0.72和RND的0.65。
2. **动作条件预测潜变量关键作用**：消融实验表明，使用预测潜变量（zp）比使用隐藏潜变量（zh）效果更好（LIBERO-Long上ROC-AUC 0.89 vs 0.85；ManiSkill-Long上0.83 vs 0.81）。
3. **跨策略泛化可行但不对称**：在真实场景中，用π0.5训练可较好迁移至ACT（ROC-AUC 0.94），但反向迁移较差（0.56），因为π0.5包含恢复行为而ACT无。
4. **序列模型优于独立帧**：MLP性能接近随机（0.5-0.59），LSTM和Transformer表现更好，说明检测需要时序上下文。

## 7. 优点

- **统一且策略无关**：不依赖策略内部状态，仅用观测和动作，可应用于任意VLA策略，甚至跨策略迁移。
- **仅需最终标签**：不需要密集的故障时间标注，降低数据收集成本。
- **统计保证**：功能共形预测提供时变阈值，在成功轨迹上控制误报率，可解释性好。
- **长周期有效性**：通过动作条件潜变量捕捉长期偏差累积，实验证明在超长任务（>8000步）上明显优于既有方法。
- **全面实验**：涵盖仿真和真实机器人、多种策略和任务，消融和泛化实验充分。

## 8. 不足与局限

- **计算开销**：世界模型编码和预测延迟约183ms，虽满足重新规划间隔，但在需要高频（每控制步）检测的场景中可能受限，不利于快速闭环控制。
- **共形假设**：校准分布需与部署分布匹配，若故障模式或任务分布变化，保证可能失效。
- **跨策略泛化不对称**：当训练策略与测试策略行为差异大（如缺少恢复动作）时，检测性能下降。论文对原因分析有限。
- **实验覆盖**：虽涵盖多个基准，但真实机器人仅4组任务，且故障类型有限（多为抓取失败、放置失败），未涉及更复杂的物理交互故障（如滑动、倾倒）。消融实验未测试不同世界模型尺寸的影响。
- **依赖预训练世界模型**：V-JEPA 2-AC本身需要大规模预训练，若任务环境与预训练分布差异大，可能需要额外微调，成本增加。
- **未报告训练总时长**：仅给出迭代次数和GPU数量，未给出实际训练时间（小时），不利于复现评估。
