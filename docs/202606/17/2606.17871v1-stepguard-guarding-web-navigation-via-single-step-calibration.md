## 摘要
网页导航要求代理遵循自然语言目标，与网页交互并生成准确的答案。尽管近期进展利用了视觉语言模型和强化学习，现有方法仍因奖励错配和错误传播而面临单步脆弱性问题。为解决奖励纠缠问题，我们设计了动态双策略优化（DDPO），该策略在探索的导航优先模式与问答的答案优先模式之间动态切换，以缓解奖励冲突。为校准单步错误，我们提出了置信引导自适应导航反思（CANR）机制，该机制估计每步置信度，仅在必要时触发反思，并使用对比奖励鼓励自我纠正以校准单步不准确性。基于上述主要组件，我们最终开发了StepGuard，一种通过单步校准守护网页导航的新框架。实验表明，我们的方法显著提升了导航与答案准确性，在标准网页导航基准上达到了新的最先进性能。

## Abstract
Web navigation requires agents to follow natural language goals, interact with web pages, and produce accurate answers. While recent advances leverage vision-language models and reinforcement learning, existing methods still suffer from single-step fragility due to reward misalignment and error propagation. To tackle the reward entanglement, we design Dynamic Dual-Policy Optimization (DDPO), which dynamically switches between a navigation-first mode for exploration and an answer-first mode for question-answering to mitigate reward conflict. To calibrate the single-step error, we propose Confidence-Guided Adaptive Navigation Reflection (CANR), a mechanism that estimates per-step confidence, triggers reflection only when necessary, and uses contrastive rewards to encourage self-correction to calibrate the single-step inaccuracy. With the above as the main components, we finally develop our StepGuard, a new framework of Guarding Web Navigation via Single-Step Calibration. Experiments demonstrate that our approach significantly improves navigation and answer accuracy, setting new state-of-the-art performance on standard web navigation benchmarks.

### 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：网页导航任务中，现有基于视觉语言模型（VLM）和强化学习（RL）的智能体存在“单步脆弱性”，表现为：**(1) 奖励冲突**——导航奖励鼓励持续探索，而问答奖励鼓励提前终止并生成答案，两者在优化时相互干扰；(2) 单步决策错误传播**——一次错误动作会沿轨迹累积，最终导致导航失败或答案错误。
- **研究背景**：Web 导航需要智能体遵循自然语言指令，与网页交互并输出准确答案。尽管 VLM（如 GPT-4、InternVL）和 RL 方法（如 GRPO）取得进展，但上述两个问题使得训练不稳定、鲁棒性差。
- **整体含义**：本文提出 StepGuard 框架，通过**动态双策略优化（DDPO）** 解耦导航与问答目标，并通过**置信引导自适应导航反思（CANR）** 按需校准单步决策，从而显著提升导航成功率和答案准确性，为构建可靠的自主网页代理提供了新范式。

### 2. 方法论
#### 核心思想
- 将单步训练样本按任务类型分为“导航子集”和“问答子集”，交替应用非冲突奖励；同时引入基于置信度的自适应反思机制，仅在不确定时触发自我纠正。

#### 关键技术细节
- **动态双策略优化（DDPO）**：
  - **导航优先模式**：奖励 = 格式奖励 \(R_f\) + 导航奖励 \(R_n\)（基于专家动作或接近目标页面的距离）。
  - **答案优先模式**：奖励 = 格式奖励 \(R_f\) + 问答奖励 \(R_a\)（基于 F1 分数）。
  - 训练时根据样本类型动态切换：若样本属于问答子集 \(D_{qa}\)，则用 \(R_a + R_f\)；若属于导航子集 \(D_{nav}\)，则用 \(R_n + R_f\)。
- **置信引导自适应导航反射（CANR）**：
  - **置信估计**：计算策略分布与均匀分布的 KL 散度 \(Conf(s_t) = D_{KL}(\pi_\theta(\cdot|s_t) \| U(C_t))\)。
  - **概率触发**：\(P_{\text{reflect}}(s_t) = \exp(-\kappa \cdot Conf(s_t))\)，\(\kappa=4\)，低置信度时更可能触发反思。
  - **反思校准**：触发后，模型先生成初始响应，再通过反思提示生成修正响应；引入**反思成功奖励** \(R_{rs}\)（若修正后动作的导航奖励更高则取 1），总反思奖励 = \(R_f + R_n + R_{rs}\)。
- **总体优化**：使用 GRPO（Group Relative Policy Optimization）算法进行策略更新，在组内对奖励归一化并进行相对比较。

- StepGuard 在 WebVLN 上达到 **SR 39.83%**（+5.07% 超越 WebVLN-Net），在 WebWalkerQA 上以 3B 参数模型超越 14B/32B 模型，接近 72B 模型（Hard 子集 SR 25.38% vs. 25.83%），证明单步校准可有效缩小小模型与大模型的差距。

2. **消融实验充分**：系统验证了每个组件（DDPO、CANR、各类奖励、反思策略、超参数）的必要性，置信度量对比和校准分析增强了方法可信度。
